Conjuntos de dados utilizados no projeto
train - base original de treino disponibilizado pelo Kaggle
test -  base original de test disponibilizado pelo Kaggle

processed_train_data - base processada em python para alimentar o treinamento do modelo1
processed_test_data - base processada em python para previsão do modelo1

teste_predict - base de teste com a previsão dada pelo modelo1 para a variável alvo

full_data - base de treino e teste processadas concatenadas para treinamento e validação do modelo2

train_data_models - base de dados gerada aleatoriamente pelo R e utilizada para treinar o modelo2, além das previsões anexadas ao dataset
test_data_models - base de dados gerada aleatoriamente pelo R e utilizada para testar o modelo2, além das previsões anexadas ao dataset
